# 深色模式主按钮对比度修复设计

## 目标

修复深色模式下主按钮背景和文字同为深蓝色的问题，使登录、注册、重置密码及站内其他默认主按钮保持清晰可读。

## 根因

默认按钮使用 `bg-primary text-primary-foreground`。当前深色主题中的 `--primary` 与 `--primary-foreground` 都是深蓝色，前景与背景亮度接近，无法形成足够对比。

## 方案

- 将深色主题的 `--primary-foreground` 和 `--sidebar-primary-foreground` 改为近白色。
- 保留 Button 组件的语义化颜色类，不在登录页或组件内硬编码白色。
- 不修改描边、次要、幽灵、危险按钮和文字链接；这些样式已有独立且可读的前景色。
- 此修复对所有使用 `text-primary-foreground` 或 `text-sidebar-primary-foreground` 的界面生效，避免其他页面重复出现相同问题。

## 验收

- 深色模式下，登录、注册和重置密码主按钮使用近白色文字。
- 深色模式下，侧边栏主操作按钮也使用近白色文字。
- 游客入口和发送验证码描边按钮仍清晰可读。
- 登录方式切换、注册、忘记密码等文字链接仍清晰可读。
- 浅色模式外观不受影响。
- 前端测试、类型检查和样式检查通过。
